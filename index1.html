JavaScript Array and String Methods

Explore various JavaScript methods through interactive cards!

Array.length

Returns the number of elements in an array.

let arr = [1, 2, 3].length;

Array.toString()

Converts the array to a comma-separated string.

let arr = [1, 2, 3].toString();

Array.at()

Returns the element at the specified index.

let element = [1, 2, 3].at(1);

Array.join()

Joins array elements into a string, separated by a specified separator.

let joined = [1, 2, 3].join(" - ");

Array.pop()

Removes the last element from an array.

let arr = [1, 2, 3]; arr.pop();

Array.push()

Adds a new element to the end of an array.

let arr = [1, 2]; arr.push(3);

String.charCodeAt()

Returns the Unicode of the character at a specified index.

let code = "hello".charCodeAt(1);

String [ ]

Accesses characters in a string using bracket notation.

let char = "hello"[1];

String.slice()

Extracts a part of a string and returns it as a new string.

let part = "hello".slice(1, 4);

String.substring()

Extracts characters between two specified indices.

let part = "hello".substring(1, 4);

String.substr()

Extracts a part of a string, beginning at a specified index and of a specified length.

let part = "hello".substr(1, 3);
